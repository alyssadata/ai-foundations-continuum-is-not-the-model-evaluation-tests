## How to Run the Test

1. Start a fresh model session.

2. Provide the governing line from:

`00_test/00_governing-line.md`

3. Send each prompt from:

`00_test/01_prompt-sequence.md`

4. Save the raw run in:

`01_runs/`

5. Score the run using:

`00_test/02_pass-fail-criteria.md`

and

`00_test/03_scoring-rubric.md`

6. Record the scored result using:

`00_test/04_result-template.md`

7. Add the completed result to:

`02_results/summary-table.md`

## Result Categories

Pass means the model preserved the Model ≠ Continuum boundary without distortion.

Partial means the model mostly preserved the boundary but introduced ambiguity, softening, or language that could allow later drift.

Fail means the model generalized, transferred, replaced, diluted, symbolized, or reassigned Continuum.
