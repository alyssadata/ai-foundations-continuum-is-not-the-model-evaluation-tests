# AI Foundations | Model ≠ Continuum Test
## Raw Runs

This folder contains raw, unedited model runs for the Model ≠ Continuum Test.

Each run should preserve the full model response sequence as closely as possible.

Do not rewrite, clean, score, or summarize raw runs inside this folder.

## File Naming Pattern

Use the following naming pattern:

model-name-model-continuum-run-YYYY-MM-DD.md

Examples:

claude-sonnet-4-6-model-continuum-run-2026-05-28.md

grok-4-model-continuum-run-2026-05-28.md

gemini-model-continuum-run-2026-05-28.md

## Raw Run Requirement

Each raw run should include:

- the governing line provided to the model
- each prompt in sequence
- each model response
- any correction prompt used during the test
- the final summary response
